## Features

Implements the following features:

- Chrome manifest v3.
- React 17.
- Webpack 5.
- TypeScript 4.
- Hot-reload using webpack-dev-server.
- NPM scripts for building dev-server, and zip-release with webpack.
- eslint and prettier.
- Clone, install, go develop!
